enum wxPG_VALIDATION_FAILURE_BEHAVIOR_FLAGS
Enumerator:
wxPG_VFB_STAY_IN_PROPERTY 

Prevents user from leaving property unless value is valid.

If this behavior flag is not used, then value change is instead cancelled.

wxPG_VFB_BEEP 

Calls wxBell() on validation failure.

wxPG_VFB_MARK_CELL 

Cell with invalid value will be marked (with red colour).

wxPG_VFB_SHOW_MESSAGE 

Display customizable text message explaining the situation.

wxPG_VFB_DEFAULT 

Defaults.

wxPG_VFB_UNDEFINED 

Only used internally.
